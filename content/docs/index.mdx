## CMSC398P: Practical Cloud Computing with AWS

Welcome to the course textbook for CMSC398P! This course provides a practical introduction to cloud computing using Amazon Web Services (AWS).

## This Website

This website hosts all of the lecture content for CMSC398P in a textbook format, rather than traditional slides. The goals of using this format are to:

- Familiarize students with navigating and using documentation
- Make implementation details easy to find while doing projects
- Provide a searchable reference for AWS concepts and services

### How to Navigate

- Use the right sidebar to get to other sections of the current page
- Use the left sidebar to get to other pages of this website
- Use the searchbar (Ctrl + K) to search through page titles and content

## Key Topics

This course will cover:

- Cloud computing fundamentals and AWS infrastructure
- Identity and Access Management (IAM)
- Core AWS services (EC2, S3, VPC, etc.)
- Security and compliance in the cloud
- Global architecture and deployment strategies
- And much more!
